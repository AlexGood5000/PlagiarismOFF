Приложение для проверки текста и программного кода на плагиат. Проверка
предоставляется для полноценных текстов на русском и английском языках, а также
для языков программирования семейства СИ, а именно: C, C++, C# и Java.

Для проверки необходимо выбрать файл на компьтере и указать категорию (каталог),
к которой предварительно прописать путь в настройках. Для визуализации
прогресса представлена шкала, которая заполняется по мере выполнения рабботы.
По завершению проверки показывается процентная уникальность проверяемого файла
и список файлов, откуда были произведены заимствования. Результаты каждой проверки
записываются в Excel таблицу, которая доступна прямо из приложения.
